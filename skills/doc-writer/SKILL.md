# 文档生成 (Documentation Writer)

为指定的代码文件或整个项目生成清晰、准确的文档。

## 文档类型

- **README.md** — 项目概述、安装、使用示例
- **函数/类文档** — JSDoc 风格的参数和返回值说明
- **API 文档** — 端点、请求/响应格式
- **架构说明** — 模块职责、数据流、关键决策

## 风格指南

- 使用中文编写（面向中文开发者）
- 简洁直接，不堆砌废话
- 代码示例放在关键位置
- 目录结构用 tree 格式展示

## 执行方式

1. 读取目标文件
2. 分析代码结构和关键逻辑
3. 生成对应类型的文档
4. 写入文档文件
