# 五行生克知识库

## 一、天干地支生克路径规则

### 1. 同柱关系

- **天干可以生克地支**：天干为君，地支为臣
- **地支只能生天干，不能克天干**：若地支克天干（如甲申，申金克甲木），说明干支关系不合，天干较弱，坐支不能提供力量，但地支不会真的去克天干

### 2. 相邻两柱天干生克

- **相邻两柱**（年-月、月-日、日-时）的天干可以直接生克
- **相隔两柱**（年-日、年-时、月-时）的天干不能直接生克

#### 例外：相隔天干也能生克的情况

1. **天干相合**：如甲己相合，年柱甲木与日柱己土虽隔一柱，但因甲己合，甲木依然克己土
2. **地支关联触发**：当地支之间发生冲、合、刑、穿时，相隔两柱的天干也会发生生克
   - 例：甲申年 戊寅日，因寅申相冲，甲木克戊土；若无冲，甲木克不到戊土
   - 同理，地支相合、刑、穿，只要地支有关联性，其天干就可以产生生克
3. **地支相同**：两个天干坐在同一个地支上，会产生生克
   - 例：甲申年 戊申时，虽隔月柱日柱，但年干甲木与时干戊土坐支都是申金，甲木克戊土
4. **藏透关系（同根同源）**：
   - 天干的根源在地支，该地支就可以给这个天干提供能量
   - 例：亥中藏甲木，甲木透干到天干，无论甲木在天干什么位置，亥水在地支什么位置，亥水都可以生甲木
   - 若两个天干来自同一个地支（地支藏干多）——如寅中藏甲丙戊，寅中甲木和戊土都透到天干，则无论相隔多远，甲木克戊土

#### 例外不成立：贪合忘克/贪合忘生

- 相邻两柱，甲木本可克戊土，但甲木左边有己土合甲木（甲己合），甲木就去合己土，不去克戊土了 → **贪合忘克**
- 同理，贪合忘生原理一样
- 另一特殊情况：A 与 C 因满足条件可以生克，但中间的 B 太强大，先把 A 打倒，A 无法与 C 生克（实质是相邻两柱生克导致力量变化）

### 3. 地支之间的生克

- **相邻两柱的地支之间可以生克**，但生克并不强烈
- 地支之间主要论：**刑、合、冲、穿**

### 4. 地支发生冲合刑穿时的藏干生克

当地支发生冲合刑穿时，地支中的藏干（无论有没有透出天干）之间都会发生生克：

- 例：寅亥相合，亥中壬水生寅中甲木，寅中戊土克亥中壬水，亥中壬水克寅中丙火
- 判断哪一方向为主：**看力量**。如亥水在月令且为冬天，则亥中壬水克寅中丙火是主要方向
- **天干透出决定冲合的含义**：
  - 寅和申：寅透甲木、申透庚金 → 寅申冲表达**金克木**（庚金克甲木）
  - 寅和申：申透壬水、寅透戊土 → 寅申冲表达**土克水**（戊土克壬水）

## 二、五行本质特性（《尚书·洪范》）

| 五行 | 特性 | 核心意象 |
|------|------|----------|
| **木** | 曰曲直 | 能屈能伸，向四面八方伸展扩张，一棵树百年后可成森林 |
| **金** | 曰从革 | 质地刚硬，可作兵器以杀戮，肃杀内敛 |
| **水** | 曰润下 | 滋润万物，向下流，随地势改变形状 |
| **火** | 曰炎上 | 焚烧、炎热、光明，向上窜，火势猛烈 |
| **土** | 曰稼穑 | 含吐万物，孕育万物，安静不动如山 |

### 五行对立关系

- **金 vs 木**：金向内收敛，木向外发展。木主生发，金主肃杀
- **水 vs 火**：水润下，火炎上
- **土**：中和，不动如山，但当水变成水蒸气时也是向上的

## 三、五行生克象意

生 = 把力量给你，如母亲哺育孩子。有情，我喜欢你、给你好处、帮助你。
克 = 排斥、管束、讨厌。我对你坏的行为就是在克你。

### 按日主视角

#### 日主为金

| 关系 | 象意 |
|------|------|
| **金生水** | 由刚变柔，刚柔并济 |
| **金克木** | 克制延伸发展 |
| **土生金** | 由静生坚 |
| **火克金** | 猛烈之火把金融化成水；光明照亮金属。人本来很坚强，但受致命打击一蹶不振。火克金一般发生得比较猛烈 |

#### 日主为木

| 关系 | 象意 |
|------|------|
| **水生木** | 木得滋润，吸收知识文化 |
| **木生火** | 木的才华得到体现。火是木开出的花，树木的花是木的精华。**木火通明**为文化之象、智慧之象。木火同体，容易联系在一起，自然现象中花朵是树木开出来的，火相当于树木开的花 |
| **木克土** | 木安静老实。一方面指用规章制度管理老实人；另一方面指哄骗老实人，花言巧语骗了老实人。**尤其甲木克戊土带欺骗性质** |
| **金克木** | 木受砍伐，限制了发展 |

#### 日主为水

| 关系 | 象意 |
|------|------|
| **水生木** | 知识才能得到展示 |
| **水克火** | 黑暗笼罩光明，阴消阳，阴暗的东西把光明的东西罩住 |
| **土克水** | 水被管制。水本流动，土包围则有**牢狱之象**（坐牢、住院等） |
| **金生水** | 刚柔并济之象，得到文化、知识、本事。也可说水去清洗了金。**金白水清**格局。尤其辛金（珠宝珠玉），水洗去浮尘更美丽好看 |

#### 日主为火

| 关系 | 象意 |
|------|------|
| **火生土** | 由动变静 |
| **火克金** | 掌控金融，照耀金属，照耀别人。分辨火是燃烧的火还是照亮的光，得看力量对比 |
| **水克火** | 火被打击灭掉，被遮蔽光芒 |
| 火性 | 猛烈躁动 |

#### 日主为土

| 关系 | 象意 |
|------|------|
| **土生金** | 由静生坚，孕育智慧，好比大地长出宝石黄金 |
| **土克水** | 围困。至少存在**两个土把水夹在中间**才是包围；只有一个土则只是阻挡 |
| **火生土** | 花朵结出果实，升华 |
| **木克土** | 一盘散沙被固定在一个地方。木力量太强大会反把土克散。也有**拨云见日**之意——土在天上：戊土为雾霾，己土为云，甲乙木为风，风吹散云雾 |
