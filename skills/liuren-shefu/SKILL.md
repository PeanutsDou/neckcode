# 六壬射覆 — Liuren Divination Skill

基于 [focussash/Liuren](https://github.com/focussash/Liuren) 核心逻辑。

## 原理

六壬式盘以月将加时排天地盘，通过日干支推算四课三传，结合五行生克、十二天将、神煞体系进行解读。

## 使用方法

调用脚本进行排盘：

```bash
python scripts/liuren_divinate.py --year 2026 --month 5 --day 12 --hour 10
```

或直接问用户时间后自动排盘。

## 工作流程

### Step 1：收集信息

问用户：
- 当前日期和时间（或想占卜的特定时间）
- 想问什么事（财运/事业/感情/健康/射覆猜物）

### Step 2：运行排盘

```python
python scripts/liuren_divinate.py --year YYYY --month M --day D --hour H
```

输出示例：
```
日柱: 甲子 | 时支: 巳 | 月将: 胜光

天盘: 子->未 丑->申 ...

四课:
  一课 二课 三课 四课

课体: 元首
三传: 初传→中传→末传
```

### Step 3：解读

参照以下框架给出解读：

1. **四课分析**：天地盘生克关系
2. **课体判定**：元首/重审/知一/涉害/遥克/昴星/别责/八专/伏吟/返吟
3. **三传走势**：初传（始）、中传（中）、末传（终）
4. **天将影响**：贵人/螣蛇/朱雀/六合/勾陈/青龙/天空/白虎/太常/玄武/太阴/天后
5. **综合结论**
